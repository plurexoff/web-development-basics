Задание №6: Основы веб‑разработки

HTML • CSS • JavaScript

1. Структура страницы (HTML)

Эта страница демонстрирует базовую разметку HTML с заголовками, абзацами и списками. Ниже приведён короткий план обучения фронтенду.

План обучения фронтенду

Рекомендуемая последовательность шагов для изучения веб‑разработки:

  • HTML – структура документа, теги, атрибуты.
  • CSS – стилизация, Flexbox, адаптивность.
  • JavaScript – язык программирования для браузера.
  • Работа с DOM – изменение содержимого страницы из кода.
  • Фреймворки (React/Vue/etc.) – по желанию, после освоения базиса.

2. Оформление страницы (CSS)

Для этой страницы используется отдельный файл styles.css, где описаны:

  1. Базовая типографика и цвета.
  2. Карточная верстка контента.
  3. Кнопки с разными стилями.
  4. Адаптивность контейнера.

Попробуйте изменить стили в файле styles.css и обновить страницу в браузере.
